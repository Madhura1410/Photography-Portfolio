 📸 Photography Portfolio Website

A clean and responsive photography portfolio website built using **HTML5** and **CSS3**. This project showcases a modern landing page with a gallery of beautiful nature photographs and a simple contact section.

 🌐 Live Demo

 https://madhura1410.github.io/Photography-Portfolio/

📖 Project Overview

This website is designed as a simple photography portfolio that highlights stunning nature images in a clean and minimal layout. It demonstrates responsive web design principles and provides an attractive user experience across different screen sizes.

✨ Features

- 📱 Fully Responsive Design
- 🖼️ Beautiful Nature Image Gallery
- 🎨 Modern Landing Page
- 📷 Photography Portfolio Layout
- 🔗 Smooth Scrolling Navigation
- 📞 Contact Section
- ⚡ Clean and Minimal UI

🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Unsplash Images
